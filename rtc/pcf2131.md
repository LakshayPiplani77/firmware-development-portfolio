## Device Overview

The PCF2131 is a high-accuracy Real-Time Clock (RTC) device from NXP Semiconductors featuring calendar, alarm, timestamp, watchdog, and interrupt functionality.

**Interface:** I2C  
**Category:** Real-Time Clock (RTC)

---

## Project Summary

Developed a CMSIS-based bare-metal driver and demonstration application for the PCF2131 RTC device using the NXP MCUXpresso SDK.

The project demonstrates RTC initialization, time/date programming, alarm configuration, and register-level communication through the I2C interface.
