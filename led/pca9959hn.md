## Device Overview

The PCA9959HN is an NXP LED driver designed for high-density LED control applications.

**Interface:** I2C  
**Category:** LED Driver

---

## Project Summary

Developed a CMSIS-based bare-metal driver and demonstration application for the PCA9959HN LED driver.

The project demonstrates LED configuration, brightness control, PWM operation, and validation on target hardware.
